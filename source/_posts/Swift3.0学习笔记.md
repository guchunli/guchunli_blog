# 基础知识
## 数据类型
* let，var分别声明常量，变量
<!--more-->
* 类型标注
```
let a:Float = 3.0
```

* 显式转换类型
```
let a = "abc"
let b = 9
let ab = a + String(b)
```

* 值转换成字符串：\()

* 数组、字典：[]
```
let arr = [String]()
let arr1 = ["a","b","c"]

let dic = [String:Float]()
```

## 控制流
* if语句中的条件必须是一个布尔表达式
