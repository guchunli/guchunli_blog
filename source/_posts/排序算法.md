1.冒泡排序
```
int n = arr.length;
int temp = 0;
for (int i = 0; i < n; i++) {
    for (int j = 1; j < (n - i); j++) {
        if (arr[j - 1] > arr[j]) {
        // swap elements
        temp = arr[j - 1];
        arr[j - 1] = arr[j];
        arr[j] = temp;
        }
    }
}
```

2.选择排序
```
for (int i = 0; i < arr.length - 1; i++) {
    int index = i;
    for (int j = i + 1; j < arr.length; j++) {
        if (arr[j] < arr[index]) {
        index = j;// searching for lowest index
        }
    }
    int smallerNumber = arr[index];
    arr[index] = arr[i];
    arr[i] = smallerNumber;
}
```

3.插入排序
```
int n = array.length;
for (int j = 1; j < n; j++) {
    int key = array[j];
    int i = j - 1;
    while ((i > -1) && (array[i] > key)) {
        array[i + 1] = array[i];
        i--;
    }
    array[i + 1] = key;
}
```
